🛰️ CanSat Telemetry API Test Interface

Current Active Database

📌 Important: All operations (push data, queries, AI questions) use the current active database shown below.

1. Create New Database

Create a new database in databases/ directory. This automatically becomes the active database.

2. Push Telemetry Data

Insert a new telemetry row into the current active database

3. Get All Telemetry

Retrieve all telemetry records from the current active database

4. Ask Question (AI)

Ask a natural language question about data in the current active database

5. Execute SQL Query

Run a custom SELECT query on the current active database
